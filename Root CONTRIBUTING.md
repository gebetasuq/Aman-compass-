# Contributing to Aman Compass

Thank you for your interest in contributing to Aman Compass! We welcome contributions from developers, researchers, designers, and government partners.

## How to Contribute

### 1. Reporting Bugs
- Check if the bug has already been reported in [Issues](https://github.com/gebetasuq/Aman-compass-/issues).
- If not, open a new issue with a clear title and description, including steps to reproduce, expected behavior, and screenshots if applicable.

### 2. Suggesting Features
- Open an issue with the label `enhancement`.
- Describe the feature, its use case, and how it aligns with the project’s goals.

### 3. Code Contributions
- Fork the repository.
- Create a new branch (`git checkout -b feature/your-feature`).
- Write clear, tested code following the existing style.
- Commit with descriptive messages.
- Push to your fork and open a pull request.

### 4. Translations
- We welcome Arabic translations and improvements. See `citizen-app/lib/l10n/` for ARB files.

### 5. Documentation
- Improvements to `docs/` or inline comments are always appreciated.

## Code of Conduct
Please note that this project adheres to a [Code of Conduct](CODE_OF_CONDUCT.md). By participating, you agree to uphold its terms.

## Development Setup
Refer to the main [README.md](README.md) for setup instructions.

## Questions?
Feel free to reach out via [GitHub Issues](https://github.com/gebetasuq/Aman-compass-/issues) or contact the maintainers directly.

Thank you for helping make Aman Compass better! 🇦🇪
